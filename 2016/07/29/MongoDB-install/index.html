MongoDB install

2016-07-29

  1. 创建data,log目录
    db: C: log: C:data.log

  2. 创建windows服务
    mongod –logpath C:.log –logappend –dbpath C:–directoryperdb –serviceName MongoDB –install

  3. 启动 net start MongoDB
