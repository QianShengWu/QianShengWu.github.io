在机器学习领域中，梯度下降的方式有三种，分别是：批量梯度下降BGD、随机梯度下降SGD、小批量梯度下降MBGD。

二维空间下：

  1. 批量梯度下降BGD

    批量梯度下降法是在更新每一参数时都使用所有的样本来进行更新。

    \[ \theta _{j}^{'} = {\theta _j} + {\frac{1}{m}}\sum_{i = 1}^m {({y^i} - {h_\theta }({x^i}))x_j^i} \] 优点：全局最优解；易于并行实现；

    缺点：当样本数目很多时，训练过程会很慢。

    从迭代的次数上来看，BGD迭代的次数相对较少。其迭代的收敛曲线示意图可以表示如下：

  2. 随机梯度下降SGD

    随机梯度下降是通过每个样本来迭代更新一次，问题是噪音较BGD要多，使得SGD并不是每次迭代都向着整体最优化方向。

    改写损失函数如下： \[ J(\theta ) = {\textstyle{1 \over m}}\sum\limits_{i = 1}^m {\cos t(\theta ,({x^i},{y^i}))} \]

    \[ \cos t(\theta ,({x^i},{y^i})) = \frac{1}{2}{({y^i} - {h_\theta }({x^i}))^2} \]

    更新\[\theta\] \[ \theta _j^{'} = {\theta _j} + ({y^i} - {h_\theta }({x^i}))x_j^i \] 优点：训练速度快；

    缺点：准确度下降，并不是全局最优；不易于并行实现。
